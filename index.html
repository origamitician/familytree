How to use:

This family tree is hand-coded without any use of website builders. It is interactive, meaning if you click on a certain name, a popup would come up that describes key things about that particular family member. Some text in the popups may be encrypted because they contain very personal details.

The "change color" commands still don't work. There also are a few visual glitches that I'm still working to fix on, but you get the general idea!

To view a cleaner family tree, go to this link: https://www.dnaweekly.com/tools/family-tree-maker/3095b483a241a3c053749d8960b75388/.
